```md
# Full-Stack Security Alert Demo

This demo shows how to detect security anomalies across the frontend, backend, database, and DevOps pipelines using lightweight tools and scripts.

## Features
- Frontend JS error tracking
- Backend suspicious input detection
- MongoDB audit logging
- GitHub secret scanning

## Getting Started
1. Clone the repo
2. Run backend: `node backend/server.js`
3. Open `frontend/index.html`
4. Push a commit with `.env` to test secret alert
```
